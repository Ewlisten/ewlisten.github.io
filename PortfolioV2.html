ML Player Stat Prediction

Cosine similarity on five-stat vectors surfaces comparables most scouts never consider together.

A scikit-learn pipeline that extracts player feature vectors, scores similarity across the league using cosine distance, and overlays two players' profiles on a radar chart. "Can a model learn to see what analysts spend hours watching?"

Object Detection · Tracking

Basketball Motion Tracking

YOLO + SORT combination achieves stable multi-object tracking across fast, chaotic game footage.

A full computer vision pipeline built for sports footage — YOLO detects objects frame by frame, SORT assigns and maintains IDs across frames, and the output is annotated video with bounding boxes and tracking trails. This is the foundation for any automated game-film analysis system.
